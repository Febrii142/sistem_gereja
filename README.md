# Sistem Pengolahan Data Gereja

Aplikasi web untuk sistem manajemen gereja dengan antarmuka yang sederhana dan mudah digunakan.

## Fitur
- **Dashboard Utama** - Ringkasan statistik dan kegiatan mendatang
- **Manajemen Data Jemaat** - Pengelolaan data anggota jemaat
- **Jadwal Ibadah dan Kegiatan** - Penjadwalan ibadah dan kegiatan gereja
- **Keuangan/Persembahan** - Pencatatan pemasukan dan pengeluaran

## Cara Menggunakan

1. Clone repository ini:
   ```bash
   git clone https://github.com/Febrii142/sistem_gereja.git
   ```

2. Buka file `index.html` di browser Anda

Tidak diperlukan instalasi atau konfigurasi tambahan. Aplikasi ini menggunakan HTML, CSS, dan JavaScript murni tanpa dependencies.

## Struktur Folder
```
sistem_gereja/
```
